# Highway Motors

Highway Motors is a ReactJS frontend website for a car dealership.

## Pages
- Home
- Cars
- Services
- About
- Contact

## Technologies Used
- ReactJS
- Vite
- CSS
- Git
- GitHub

## How to Run
1. Download the project
2. Open the folder in VS Code
3. Run `npm install`
4. Run `npm run dev`

## Description
This website helps users view car options, dealership services, and contact information.
